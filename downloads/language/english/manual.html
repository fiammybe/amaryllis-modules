Downloads Manual

Downloads is a small, light weight file download handling module. You can upload a file, describe it and add some images (currently 4 screenshots) to present the key functionalities. Also you have the posibility to link the module with 'Album' module and/or 'Catalogue' module. Here are some instructions how to use 'Downloads' module.

First Start

There are some things you should do before you try to upload a new file. Please, follow the instructions.

Allow the necessary Filetypes

  • Go to System > Media > Mime Types
  • Choose the Mimetypes you want to allow for File upload. If you like to use fileupload from frontend beware of the group permissions per file.
  • Remember the image upload, too. Actually the mime type permissions are used by the image uploaf, too. So please allow also jpeg, jpg, png and whatever you like to use.
  • Done! Now you can upload all filetypes.

Check out the module preferences

  • Go to Modules > Downloads > Preferences
  • Choose the preferences like prefered
  • Done! Next step.

Create a category

Go to Tab 'Manage categories' > Add Category. Just Fill out the fields. Done.

Upload Files

Now you can start uploading files. Go to Tab 'Manage Files' and fill out the fields. If you have allowed using the album module you will see a field where you can select an album. All the images from the Album will be shown. You like to upload images too? Well, so use the image upload to add more images.

If you have allowed using catalogue module, the result is dependent on catalogue settings. If you have allowed to show prices, you will see a Price on two Places in Frontend: behind the title of the file and in the information-box. Mirrors are not visible for catalogue items and the download link will redirect to the catalogue item to buy it.

Requirements and Keyfeatures of the file are displayed in a list

Thank you for using 'Downloads' module. If you have some trouble while using downloads module, please visit ImpressCMS Community Forum

Icmspoll Manual

Icmspoll é um módulo de pesquisa para ImpressCMS, que podem facilmente ser usados ??com iForum, o módulo de fórum de ImpressCMS também. Ele foi projetado para fornecer as pesquisas dos utilizadores e / ou sondagens de visitantes.

Keyfeatures

  • Você pode criar enquetes com multiplas opções de escolha
  • As opções podem ser selecionadas usando radio buttons ou checkboxes, o que for melhor para sua pesquisa
  • Saída diferente dos resultados para administradores e usuários do módulo simples / convidados
  • Use o sistema de permissão do módulo para selecionar quais grupos podem ver a enquete
  • Use o sistema de permissão do módulo para selecionar quais grupos podem votar a enquete

Iniciar Primeiro

Existem diversas coisas a serem feitas antes de você criar uma pesquisa. Por favor siga as instruções

Check as preferências do módulo

Existem algumas preferências que você pode querer alterar antes de iniciar o módulo. Vá em Modulos > Icmspoll > Preferencia

  • Permissão Grupo
    • As permissões do grupo são apenas para definir as permissões para a criação de novas enquetes. Todos os grupos selecionados terão permissão para apresentar novas pesquisas.
  • Formatar Data
    • O formato da data irá ser usada para exibir a data em toda a interface do módulo. Você precisa de ajuda para obter o formato de data preferido? Veja aqui detalhes e instruções.
  • Restrição por IP
    • Configura a restrição de voto por endereço IP.
  • Restringir por impressão digital da sessão
    • Defina as restrições para as pesquisas de voto por impressão digital da sessão.
  • Restrição por usuário
    • Defina as restrições para as pesquisas de voto por identificação de usuário.
  • Mostar caminho da pagina
    • Selecione "SIM" para exibir o caminho no topo da página, "Não" para desabilitar.
  • Exibir enquetes
    • Definir o limite para as pesquisas a serem listadas na visualização do índice. "0" desliga-se a limitação e busca todas.
  • Pesquisa em Ordem padrão
    • Definir a ordem padrão para as enquetes sobre visualização do índice.
  • Enquete de classificação padrão
    • Defina o tipo padrão para as enquetes sobre visualização do índice.
  • Permitir valor inicial?
    • Selecione "SIM" para permitir valor inicialpara as opções
  • Imprimir Rodapé
    • Adicionar um rodapé de impressão para os layouts, se necessário
  • Imprimir logo
    • Você pode definir o logo a ser impresso. Basta digitar a url do caminho para o logotipo. ex.: /themes/mytheme/images/logo.gif
  • use RSS Feeds
    • Se você gosta de fornecer um link RSS e usar o recurso de RSS de "Icmspoll", definir como yes.
  • Limit of RSS Feeds
    • Definir o limite de pesquisas para RSS feeds. "0" desativará RSS-Limit.

Criando uma enquete

O próximo passo importante será a criação de uma enquete. Caso contrário, você não conseguirá visualizar a página de apuração. Ir para "enquetes"> Adicionar. Basta preencher os campos.

  • Questões
    • A questão é usado como um título e fornece também o link para a votação em exibição.
  • Descriçõa
    • A descrição pode ser usado, para descrever a questão mais detalhado e será exibido na tela de índice como em exibição simples na enquete.
  • Selecione o delimitador a ser usado
    • O módulo oferece atualmente dois delimitadores: a <br />-Tag e um espaço. Os delimitadores são usados ??para os campos de opções dentro do formulário.
  • Inicio da Enquete
    • Este será o horário de início, quando a votação é visível. Lembre-se do tempo que você precisa adicionar opções para a enquete!
  • Fim da Enquete
    • O fim do período para votação
  • Exibir em bloco
    • Defina SIM, se a enquete deve ser exibida no "Bloco Enquetes recentes"
  • Peso
    • Digite peso para ordem de exibição da enquetes.
  • Permitir multiplas seleções
    • Defina SIM, se vc quiser permitir que o participante slecione mais de uma opção.
  • Notificar o autor quando a enquete expirar?
    • Será enviado uma notificação para o criador da enquete, se a enquete expirou.
  • Exibir Permissões
    • Defina os grupos que possuem permissão para visualizar a enquete
  • Permissões de Voto
    • Defina os grupos que possuem permissão para votar na enquete

Pronto! Após o envio, vc deve ser direcionado para a página de opções, e ser capaz de adicionar opções para sua enquete.

Criando opções

O formulário de opções é muito curto. Basta digitar o texto para a opção, escolher uma cor como fundo e inserir um peso, se desejar. Antes de salvar a opção, certifique-se ter selecionado a enquete correta.

Blocos

Atualmente estão disponívies 4 blocos para o módulo:

  • Enquetes Recentes
    • O bloco "Enquetes Recentes" como o próprio nome diz exibirá as ultimas enquetes conforme a classificação definida : ordem publicação/expirado.
  • Enquete Única
    • este bloco irá exibir uma unica enquete,que pode ser posicionda em qualquer lugar.
  • Resultados Recentes
    • Este bloco irá exibir enquetes expiradas,com um link para a página de votação.
  • Unico Resultado
    • Este bloco irá exibir o resultado de uma única enquete,com dados diferentes para clientes/usuários e administradores do módulo.

Você pode conar todos os blocos, se precisar.

Icmspoll Manual

Icmspoll is a poll module for ImpressCMS, which can easily be used with iforum, the forum module of ImpressCMS, too. It's designed to provide user polls and/or visitor polls.

Keyfeatures

  • You can create polls with multiple options
  • Your options can be selected by using radio buttons or checkboxes, whatever is needed for your poll
  • Different output of results for module admins and simple users/guests
  • Use core permission system to select, which groups can view the poll
  • Use core permission system to select, which groups can vote the poll

First Start

There are several things you should do before you try to create a new poll. Please, follow the instructions.

Check out the module preferences

There might be some preferences, you'd like to change before using the module first time. Go to Modules > Icmspoll > Preferences

  • Group Permissions
    • The group permissions are only to set permissions for creating new polls in front end. All selected groups will have permissions to submit new polls.
  • Date Format
    • The Date format will be used for displaying the date throughout the module frontend. You need help to get the preferred date format? Look here for a detailed instruction.
  • Restricted by IP
    • Set the restrictions for polls voting by IP Address.
  • Restricted by Session Fingerprint
    • Set the restrictions for polls voting by Session fingerprint.
  • Restricted by User
    • Set the restrictions for polls voting by User-ID.
  • Show Breadcrumb
    • Select "Yes" to display the breadcrumb in front end, "No" to disable the breadcrumb.
  • Show polls
    • Set the limit for polls to be listed on index view. "0" will turn off the limitation and fetch all polls.
  • Polls default order
    • Set the default order for polls on index view.
  • Polls default sort
    • Set the default sort for polls on index view.
  • Allow initial values?
    • Set "YES" to allow initial Values for options
  • Print footer
    • Add a print footer for print layouts, if needed
  • Print logo
    • You can define a logo to be printed. Just enter the url from root to the logo. e.g.: /themes/mytheme/images/logo.gif
  • use RSS Feeds
    • If you like to provide a RSS link and use the RSS feature of "Icmspoll", set to yes.
  • Limit of RSS Feeds
    • Set the limit of polls for RSS feeds. "0" will disable RSS-Limit.

Create a poll

The next important will be creating a poll. Otherwise you can not see the page to submit polls. Go to Tab 'Polls' > Add poll. Just Fill out the fields.

  • Question
    • The question is used like a title and provides also the link to the poll on index view.
  • Description
    • The Description can be used, to describe the question more detailled and will be displayed on index view as on single poll view.
  • Select the delimeter to be used
    • The module provides currently two delimeters: the <br />-Tag and a Space. The delimeters are used for options fields inside the form.
  • Start of poll
    • This will be the start time, when the poll is visible. Remember the time you need to add options to the poll!
  • End of poll
    • The of the poll.
  • Display in Block
    • Set to yes, if the poll should be visible in "recent polls block"
  • Weight
    • Enter weight to order polls by weight.
  • Allow multiple selection
    • Set to Yes, if you want to have checkboxes instead of radio buttons in poll form
  • Notify the poll author when expired?
    • This will send a notification to the creator of the poll, if poll has expired.
  • View permissions
    • Set the groups to be allowed to see the poll
  • Vote permissions
    • Set the groups to be allowed to vote the poll

Done! After submitting you should directly be redirected to the options page and be able to add options to your poll.

Create Options

The options form is very short. Just enter the text for the option, select a color as background and enter a weight. Before submitting the option, be sure, the correct poll is selected.

Blocks

Currently you have 4 blocks for 'Icmspoll' module:

  • Recent Polls
    • The "Recent Polls" block will display the recently submitted polls. Select sort/order/expired polls/or publisher of the polls to get a more detailed result.
  • Single poll
    • This block will display a single poll, which can be voted anywhere.
  • Recent results
    • This block will display recent expired polls with a link to the poll
  • single result
    • This block will display a single result, different data for guests/users and module admins.

You can clone all blocks, if needed.

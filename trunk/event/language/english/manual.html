Event Manual

Event is a simple event-handling module for ImpressCMS which can also display some public rss calendars

Keyfeatures

  • Create fast and simple an event and categorize them using different categories/colors
  • Categories are using the group permissions of ImpressCMS to allow different Groups to view categories and it's Events
  • Categories have a group permission for submitting new Events to handle Groups and different submit permissions
  • Import public calendars simple by using RSS-Feeds (e.g. public google calendars)

First Start

Create First Category

Creating categories is simple. There are not really much fields.

  • Title will be used for Calendar Legend
  • Description will be hovered in a qtip-span in Legend
  • Color will be used as a background Color of the events displayed in calendar
  • Textcolor will be used as a text Color of the events displayed in calendar. Default would be black. But in some darker colors it might be better to adjust the text-color

That's it. After setting the permissions you can go forward and add some events to your category

Templates

There're just a few templates available:

  • /event_admin.html
    • contains all necessary variables to display admin-side. Less informations enough, and I think, most people will never use this template for changes ;-)
  • /event_requirements.html
    • The same: only an admin template and will only be used, if the requirements check failed on installation.
  • /event_header.html & /event_footer.html
    • Header and footer holding additional informations like RSS-Feeds, Header informations, comment loop, notifications
  • /event_index.html
    • Currently displays the calendar and whatever needed for
  • /event_event.html
    • Dipsplaying full event Informations for a single event
  • /event_calconf.html
    • Is holding the calendar jquery configuration of the calendar

Event Modul Anleitung

Das Event Modul ist ein einfaches Veranstaltungs Modul für ImpressCMS. Es kann auch in dem öffentliche rss Feed Kalender angezeigen.

Funktionen

  • Erstellen Sie schnell und einfach ein Veranstaltungen und kategorisieren sie diese mit verschiedenen Farben für die Kategorien
  • Mit Hilfe der Gruppenberechtigungen von ImpressCMS können Sie festlegen welcher Kalender für welche Gruppe sichtbar ist.
  • Ebenfalls können Sie festlegen Welche Benutzergruppe je Kalenderkategorie die Berechtigung haben neue Veranstaltungen einzureichen.
  • Importieren von öffentlichen Kalendern einfach unter Mithilfe der RSS-Feeds (zB. Ihre öffentlichen Google Kalender)

Einstellungen

Die Modul Einstellungen kurz und knapp in der übersicht:

  • Standard Ansicht
    • Die Standard-Ansicht definiert, welche der verfügbaren Ansichten aktiv sind, wenn Sie das Modul aufrufen. "Monat" zeigt den aktuellen Monat, "Woche" zeigt die aktuelle Woche und "Tag" der aktuelle Tag
  • Der erste Tag in der Woche angezeigt
    • Wählen Sie, welcher Wochentag als erster Tag in der Ansicht erscheinen soll Montag oder Sonntag?
  • Wochenend Ansicht
    • Wählen Sie, ob das Wochenende im Kalender angezeigt werden soll?
  • Startzeit des Kalenders
    • Die Startzeit ist die Zeit,die in der Tagesansicht um zuerst zu sehen sein soll. Wenn der Kalender skaliert wird (abhängig vom Thema was Sie verwenden) werden alle Stunden davor nicht angezeigt. So wird der Kalender bei der vorgegebenen Stunde in der Tagesansicht beginnen.
  • Min. Zeit für Woche / Tag Ansicht.
    • Hier können Sie die minimale sichtbare Stunde. Z.B. auf 6 Uhr, alle Stunden davor sind nicht sichtbar im Kalender. Wenn Sie ein Ereignis festlegen, bevor dieser Zeit, kann man es in der Monatsansicht sehen
  • Max Zeit für Woche / Tag Ansicht.
    • Die letzte sichtbare Stunde.
  • Zeiteinteilung in der Minuten in Agenda-Ansicht
    • Die Zeiteinteilung Minuten, definiert wie die Stunde aufgeteilt werden soll in Tagesansicht (z.B. 30 Minuten Takt )
  • Die folgenden Datumsformate gemeinsam abgewickelt werden kann:
    • Behandeln Sie Datum und Uhrzeit Formatierung für Kalender ausgegeben. Bilden weitere Informationen zu überprüfen Datumsformat in Event Module
  • Verwenden Sie die UI Theme
    • Standardmäßig im Kalender ist das ui Thema enthalten. Um den Stil zu ändern, haben Sie zwei Möglichkeiten:
      • Suche nach einem anderen Thema auf jQuery UI Homepage (dadurch werden sich alle ui Stile auf der Website zu ändern oder besuchen Sie ui Thema roller)
      • Schalten Sie das UI Theme aus und gestalten Sie Ihren individuellen Stil. Dies kann auch geschehen, während ui-Thema ist aktiviert ist, es ist aber nicht so einfach ,mit ui Thema im Hintergrund Ihren Kalender zu gestalten
  • Use a checkbox-Form to select, which Categories/Calendars should be loaded?
    • The module supports a checkbox form, to select, which calendars currently should be displayed. Just check/uncheck all calendars to be loaded. Currently this does not properly work in other browsers than FF..!
  • Print footer
    • Add a custom footer to print layout
  • Print logo
    • Add a custom logo to print layout. The path needs to be as default. 'themes/iTheme/img/logo.png'. From root path of your ImpressCMS Install to the logo.
  • Profile Module Settings
    • The following both fields require profile module to be installed. If Profile Module is available, you can add a "birthday" field to your user profiles. Let's call it "birthday" here in example, but you can call it however you like. So, go to profile module and add a new field with type "date" and give a field name "birthday" and Caption "Your Birthday". If you like, set as required and a step for registration or even let it not required. Doesn't matter. If the user has added his birthday, event module can add the birthdays itself to events. Just enter the given name (NOT Caption!) here in Birthday Field and select the Category. If your added category is not present in the list, please make an update of the module. ImpressCMS does not catch the full list of categories without updating the module! If the profile Module is installed and available, the module checks (using Auto Tasks) in a given frequenzy for new user birthdays. As default I set up all 6 hours. If you don't need or just need once per day go to autotasks and disable or change the frequency as needed. All done? Well, now the module checks all time for birthday events in the current and the following year.

Füllen Sie Ihren Kalender

Die Formen des Event-Modul alle sollten klein und einfach genug. Nur ein paar Worte um, bevor Sie starten:

  • Der Kalender Tab ist nur erforderlich, wenn Sie Kalender über JSON-Feed, wie Google Kalender machen möchten. Um mehr über Howto Hinzufügen von Google Kalender lesen Sie unter Google Kalender in Event
  • Die Kategorien werden verwendet, um verschiedene Farben / Text Farben für Events zu erleichtern, um die gesuchte Art von Element zu finden definieren. Google Kalender werden mit der gleichen Funktion. Für Kategorien ist es möglich, sich zu unterwerfen und zu sehen Berechtigungen festzulegen. Alles erlaubt "view"-Gruppen können die Ereignisse innerhalb der Kategorie. Die "Eintragen" - Gruppen können Ereignisse an der Kategorie einreichen
  • Die Ereignisse haben ein Feld für Ansprechpartner. Als Standard werden Sie Ihren Account-Namen und E-Mail in ihren Bereichen zu finden. Wenn der Kontakt Name ist ein anderer Benutzer, und Sie geben Sie den Benutzernamen, Sie werden die Benutzer mit avatar bekommen. Ansonsten in der Name und die E-Mail eingeben. Wenn die E-Mail an gravatar angemeldet ist, sollten Sie den Avatar für diese E-Mail. E-Mail Adressen sind für registrierte Nutzer, nicht für die Gäste.

Google Kalender im Veranstaltungmodul

Mit dem Event-Modul haben Sie die Möglichkeit, den öffentlichen Google Kalender in Ihren kalender auf der Webseite einzubinden. Vor der Einbindung des Google Kalenders, überprüfen Sie Folgendes:

  • Sie müssen zuerst Ihren Google Kalender öffentlich freigeben machen. Oder einen neuen Kalender erstellen und dann öffentlich freigeben.
    • Melden Sie sich in Ihren Google Account an. Dann werden Sie in Ihrem Kalender auf der linken Seite zu finden
    • Klicken Sie auf den Pfeil auf der rechten Seite neben dem Kalender, den Sie freigeben möchten, danach klicken Sie auf "Share-Kalender"
    • Stellen Sie sicher, "Share nur meine Frei / Gebucht-Informationen" aktiviert ist. Der Kalender muss nur Daten zeigen. Changings durch Event-Modul oder das Hinzufügen von Ereignissen ist derzeit nicht möglich.
    • Klicken Sie als nächstes auf Kalender Einstellungen (des freigegebenen Kalenders) und überprüfen Sie die URL. In der aktuell geöffneten Feld Einstellung finden Sie die Adresse des Kalenders
    • Klicken Sie auf die Taste, um die URL, die Sie brauchen, um den Kalender in Ihrem "Kalender" Tab hinzufügen in Event-Modul bekommen.
    • Schauen Sie sich die Time-Zone-Einstellungen auf der gleichen Seite. Die Zeitzone müssen Sie in Ihrem Kalender eingestellt werden, wenn Sie das Hinzufügen Ihrer Google Kalender verwenden. Beispiele für Zeit-Zonen sind: "Europa / Berlin", "Amerika / Chicago", "America / Santo_Domingo", "Europa / London" und viele mehr. überprüfenPHP.net ZeitzoneWenn Sie nicht wissen, die benötigte Zeitzone
      • Es gibt eine Menge von nützlichen öffentlichen Kalendern Google Gruppen die Sie nutzen können. Zum Beispiel können Sie gerne Wochennummern oder Feiertage hinzuzufügen.

      https://www.google.com/calendar/feeds/de.german%23holiday%40group.v.calendar.google.com/public/basic for (german) holidays

      https://www.google.com/calendar/feeds/e_2_de%23weeknum%40group.v.calendar.google.com/public/basic for week numbers

Datumsformate im Veranstaltungs Modul

Nun lassen Sie uns die seit Einstellungen für Kopf-und Column. Auf den ersten lasst uns sicher, wir sprechen die gleiche reden:

In beiden Fällen verwenden Sie die folgenden Möglichkeiten, um die Kopfzeile oder die Spalte einzustellen wie Sie es gerne möchten:
  • s - Sekunden
  • ss - seconds, 2-stellig
  • m - Minuten
  • mm - Minuten, 2-stellig
  • h - Uhrzeit, 12-Stunden Format
  • hh - Stunden, 12-Stunden Format, 2-stellig
  • H - Stunden, 24-Stunden Format
  • HH - Stunden, 24-Stunden Format, 2-stellig
  • d - Datum
  • dd - Datum, 2-stellig
  • ddd - Datum , kurz
  • dddd - Datum, vollständig
  • M - Monat number
  • MM - Monat number, 2-stellig
  • MMM - Monatname, Kurz
  • MMMM - Monatsname, vollständig
  • yy - Jahr, 2-stellig
  • yyyy - Jahr, 4-stellig
  • t - 'a' or 'p'
  • tt - 'am' or 'pm'
  • T - 'A' or 'P'
  • TT - 'AM' or 'PM'
  • u - ISO8601 format
  • S - 'st', 'nd', 'rd', 'th' for the date

Und einige Sonderzeichen:

'...'
wörtlichen Text
''
Apostroph (vertreten durch zwei einfache Anführungszeichen)
(...)
zeigt nur Format, wenn eine der eingeschlossenen Variablen nicht null

Module Blöcke

Derzeit hat das Modul 4 Blöcke:

  • Mini Kalender Block
    • Der Block zeigt nur einen kleinen Monatskalender. Alle Tage sind mit der Tagesübersicht Ansicht verknüpft, damit ist einfach von überall aus zu einem bestimmten Tag im Veranstaltungskalender zu gelangen. Der Kalender hat keine weiteren Optionen, um (noch) nicht und wenn man es einstellen mögen: Der Mini-Kalender wverwendet jQuery UI Datepicker Script, Wenn Sie dem Link folgen, erfahren Sie mehr wie Sie den Kalender anpassen können
  • Veranstaltungsliste
    • Dieser Block zeigt nur eine einfache Veranstaltungs-Liste, welche im Kalender eingetragen ist. Sie haben die Möglichkeit bestimmte Kategorien auszuwählen, die angezeigt werden sollen. Auch können Sie den Zeitbereich skalieren.Beispiel wählen Sie keine Kategorie und aber den Zeitbereich "aktueller Monat" dann werden nur Veranstaltungen des aktuellen Monats in der Liste angezeigt. Die nachstehenden Zeitbereiche sind vorhanden:
      • Aktueller Tag: Anzeige aller Veranstaltungen des laufenden Tages
      • Tag: Anzeige aller Veranstaltungen der nächsten 24 Stunden
      • Aktuelle Woche: Anzeige aller Veranstaltungen der laufenden Woche
      • Woche: Anzeige aller Veranstaltungen für eine Woche
      • Aktueller Monat: Anzeige aller Veranstaltungen des laufenden Monats
      • Monat: Anzeige aller Events für einen Monat
  • Event Filter
    • This block shows just a simple Event List, the same as "Event List" . You can select the category to be displayed and check some additional filters. So for examle select no category and apply the other filters. Now you should see in the block on front end a form inside the block. If a category is selected, there's no category filter. Otherwise you get a select box for categories, too. If you change the start or end-range or select inside any other filter, the block will be updated automatically. Current Filters:
      • Start: Display all Events which are starting on this day or later.
      • End: Display all Events which are starting on this day or earlier.
      • Category:If no category has been selected, you get a select box. otherwise the block is searching only in the pre-selected category
      • Postal Code: Display all Events for this postal code
      • City: Display all Events for this city
  • Category/Calendar List
    • This Block just returns the legend of calendars/categories with a link to the requested category/calendar. Clicking the link, just this category will be displayed.

"Event" Module playing together with "profile" Module

The module can/will play together with profile Module. As you have recognized in Preferences the module can feth users birthdays from User Profiles and add them as an Event into your calendar. This can be done for the current year and the next year.

But it will do some more. If you allow users to join Events, they can see, which of their Friends are joining, too. This is using the profile friendship handler to get the frieds of the current user.

How to use Tools

Tools can easily setup your autotasks. Please, go to system » System Tools » Auto Tasks. There you'll find some new auto tasks. At least, you can decide yourself, if you prefer running them seperatly or even all at once by using the samled tools-autotask to run all at once to the same time.

Article Manual

Article is an article handling module for ImpressCMS.

Keyfeatures

  • You can write articles
  • You can categorize your articles
  • Attach files to your articles.
  • Use simple method to paginate your articles

First Start

There are several things you should do before you try to create a new article. Please, follow the instructions.

Allow the necessary File types

At first we need to allow the Mime types, which can be uploaded. Go to System > Media > Mime Types

Choose the Mime types you want to allow for File upload. If you like to use file upload from frontend beware of the group permissions per file.

Remember the image upload, too. Actually the mime type permissions are used by the image upload, too. So please allow also jpeg, jpg, png and whatever you like to use.

Done! Now you can upload all file types.

Check out the module preferences

There might be some preferences, you'd like to change before using the module first time. Go to Modules > Article > Preferences

Choose the preferences like preferred. Think, most preferences should be explained enough.

Done! Next step.

Create a category

The next important will be creating a category. Otherwise you can not see the page to submit articles. Go to Tab 'Manage categories' > Add Category. Just Fill out the fields. You can select an image for the category or upload a new one. You also can upload a set of new category images to /uploads/article/categoryimages/. The module will read the image files inside the folder and show them inside the select box. Done.

Submit Articles

Finally you can start creating articles. Go to Tab 'Manage Articles' > 'Add Article' and fill out the fields. Done! Now you should see your first category and article inside the frontend.

Blocks

Currently you have 4 blocks for 'Article' module:

  • Recent Articles
    • The "Recent Articles" block will display the recently submitted articles and sort them by published date, DESC.
  • Recent updated Articles
    • This block will display recently updated articles and sort them by updated date, DESC.
  • Most popular Articles
    • This block content will be dependent from your preferences. Only Articles, which where called more about the selected number will be shown inside the block.
  • Category Menu block
    • Here you c an see the category like a menu. You also can select one special category to display all subcategories of this one.
  • Random Articles block
    • Here you can see shuffled articles. You also can select one special category to display all articles of this one.
  • Article Spotlight block
    • Here you can see latest published articles in a spotlight to be displayed on a main page. You also can select one special category to display all articles of this one.

You can clone all blocks, if needed.

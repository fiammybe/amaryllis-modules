Icmspoll Manual

Icmspoll ist ein Umfrage-Modul für ImpressCMS, welches einfach mit iforum, dem Forum-Module von ImpressCMS, genutzt werden kann. Es wurde erstellt um einfach Umfragen für Besucher und Gäste bereit zu stellen.

Kern-Merkmale

  • Es können Umfragen mit beliebig vielen Optionen erstellt werden
  • Umfragen können sowohl für eine einfache als auch eine multiple Auswahl an Optionen erstellt werden
  • Unterschiedliche Ergebnisse mit Druck-Möglichkeit für Administratoren/Besucher
  • Benutzereinschränkungnen zum sehen einer Umfrage sowie zum Abstimmen einer Umfrage, seperat gehandhabt.
  • RSS-Feeds: Benachrichtigen Sie einfach ihre Stammgäste über neue Umfragen

Erster Start

Es gibt einige Kleinigkeiten, die Sie checken sollten, bevor Sie anfangen, neue Umfragen zu erstellen. Bitte, folgen Sie den Instruktionen.

Checken Sie die Modul-Einstellungen

Evfentuell gibt es einige Einstellungen des Moduls, die Sie angleichen möchten, bevor Sie das Modul nutzen. Gehen Sie zu Module > Icmspoll > Einstellungen

  • Erlaubte Gruppen zum erstellen von Umfragen
    • Wähle die Gruppen, die eine Berechtigung zum erstellen von Umfragen haben sollen. Alle gewählten Gruppen werden berechtigt sein, neue Umfragen im Front end zu erstellen.
  • Datumsformat
    • Das gewählte Datumsformat wird durch das gesamte Modul hinweg genutzt, um das Datum darzustellen. Sie benötigen Hilfe, das gesuchte Format zu finden? Schauen Sie bitte hier für detaillierte Instruktionen.
  • Eingeschränkt nach IP
    • Schränke das Wählen nach IP Addresse ein. Jede IP kann nur einmal wählen.
  • Eingeschränkt nach Session Fingerprint
    • Schränke das Wählen nach Session fingerprint ein. Der Session Fingerprint wird durch ICMS erstellt und in der DB abgespeichert.
  • Eingeschränkt nach Benutzer
    • Schränke das Wählen nach User-ID ein. Das heißt, jeder Benutzer kann nur einmal wählen. Vorsicht, wenn Sie Gästen erlauben wollen, abzustimmen.
  • Zeige Breadcrumb
    • Wählen Sie "JA" um die Breadcrumb im Front end zu zeigen, "Nein" um diese auszublenden.
  • Zeige Umfragen
    • Setzen sie das Limit für Umfragen auf der Indexseite. Sind mehr Umfragen vorhanden, wird eine Seitennavigation erstellt. "0" stellt das Limit ab, was bedeutet, dass alle Umfragen gelistet werden.
  • Ordne Umfragen standardgemäß nach
    • Wählen Sie, nach welchem Feld die Umfragen Standardgemäß im Index sortiert werden sollen.
  • Standartgemäße Sortierung
    • Wählen Sie, ob die Umfragen im Index aufsteigend oder Absteigend sortiert werden sollen.
  • Erlaube Anfangswerte?
    • Wählen Sie "JA" um Anfangswerte für Optionen zu erlauben. Damit kann ein Wert gesetzt werden, der in den vorläufigen Ergebnissen mit eingerechnet wird.
  • Print footer
    • Schreiben Sie einen print footer für print layouts, wenn Sie einen benötigen
  • Print logo
    • Sie können ein Logo festsetzen, welches zum Druck verwendet wird. Geben Sie einfach den Pfad von root zum Logo. z.B.: /themes/mytheme/images/logo.gif
  • Nutze RSS Feeds
    • Wird diese Einstellung auf "JA" gesetzt, wird auf der Index-Seite ein RSS-Link bereit gestellt und RSS-Feeds werden erlaubt
  • Limit für RSS Feeds
    • Setzen Sie das Limit für RSS-Feeds. "0" wird das RSS-Limit aufheben. Das bedeutet, alle Umfragen werden geholt.

Erstelle eine Umfrage

Der nächste Schritt wäre nun endlich das Erstellen einer Umfrage. Gehe zu dem Tab 'Umfragen' > Erstelle Umfrage. Einfach die Felder ausfüllen.

  • Frage
    • Die Frage wird wie ein Titel genutzt und wird auch zum Bereitstellen eines Links zu der Umfrage verwendet.
  • Beschreibung
    • Die Beschreibung kann genuzt werden, um die Frage detaillierter zu beschreiben und wird mitsamt der Frage und den Optionen dargestellt
  • Wähle das Trennzeichen
    • Das Modul unterstützt derweil 2 Trennzeichen: Den <br />-Tag (Zeilenumbruch) und ein Leerzeichen. Die Trennzeichen werden für die Optionen der Umfrage genutzt. Somit kann man jede Option in einer neuen Zeile oder alle in einer Darstellen.
  • Startzeit
    • Setzen Sie die Startzeit, ab welcher die Umfrage sichtbar sein soll und genutzt werden kann. Vergessen Sie nicht, die Zeit einzuplanen, um Optionen zuzufügen!
  • Endzeit
    • Das Ende der Umfrage. Ist dieser Zeitpunkt erreicht, wird die Umfrage aus der Indexübersicht genommen und taucht unter den Ergebnissen auf.
  • Zeige in Block
    • Sie können wählen, ob die Umfrage in den Blöcken erscheinen soll oder nicht
  • Gewichtung
    • Sie können eine Gewichtung angeben, müssen aber in dem Fall beachten, dass die Standart-Sortierung in den Einstellungen auch auf Gewichtung steht!
  • Erlaube multiple Optionen
    • Steht dieser Wert auf ja, bekommen Sie das Umfrage-Formular mit Checkboxen, nicht mit Radio-Buttons
  • Benachrichtigen wenn die Umfrage abgelaufen ist?
    • Sie können sich derweil via PM benachrichtigen lassen, wenn die Umfrage abgelaufen ist.
  • Berechtigung zum Betrachten
    • Wählen Sie alle Gruppen aus, welche die Umfrage sehen dürfen. Für alle gewählten Gruppen wird die Umfrage dann in der Indexseite und/oder im Block zu sehen sein
  • Wahlberechtigung
    • Wählen Sie alle Gruppen, die abstimmen dürfen. Beachten Sie, dass nur Gruppen abstimmen können, welche die Umfrage auch sehen!

Das war's! Nach dem Erstellen der Umfrage sollten Sie direkt bei den Optionen landen und können also Optionen zu ihrer Umfrage zufügen.

Erstelle Optionen

Das Formular zum erstellen von Optionen ist klein genug. Geben Sie einfach den Text der Option ein, wählen Sie eine Farbe als Hintergrund und bei Bedarf noch die Gewichtung sowie den Anfangswert, wenn gewünscht. In diesem Fall wird der gesetzte Wert als abgegebene Stimmen mitgerechnet

Blöcke

Aktuell kommt das Modul mit 4 Blöcken:

  • Letzte Umfragen
    • Dieser Block stellt eine Liste der letzten Umfragen dar. Verschiedene Einstellungen wie Sortierung/order/abgelaufene Umfragen/oder Ersteller derselben geben Anpassungsmöglichkeiten, um den Block vielfältig einzusetzen.
  • Einzelne Umfrage
    • Dieser Block stellt eine einzelne Umfrage dar, welcher überall gezeigt und abgestimmt werden kann
  • Letzte Erdebnisse
    • Dieser Block stellt eine Liste mit abgelaufenen Umfragen Bereit mit Link zum Ergebnis
  • Einzelnes Ergebnis
    • Dieser Block zeigt ein einzelnes Ergebnis einer Umfrage, mit unterschiedicher Ausgabe und Details für Gäste/Benutzer und Administratoren

Alle Blöcke können bei Bedarf geklont werden.
